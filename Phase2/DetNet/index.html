IETF

1. Deterministic Networking Architecture

Publication URL: https://tools.ietf.org/html/rfc8655

Introduction:

This document provides the overall architecture for Deterministic Networking (DetNet), which provides a capability to carry specified data flows for real-time applications with extremely low data loss rates and bounded latency within a network domain.

2. DetNet Data Plane Framework

Publication URL: https://tools.ietf.org/html/draft-ietf-detnet-data-plane-framework

Introduction:

This document provides an overall framework for the DetNet data plane. It covers concepts and considerations that are generally common to any Deterministic Networking data plane specification.

3. DetNet SRv6 Data Plane Encapsulation

Publication URL: https://tools.ietf.org/html/draft-geng-detnet-dp-sol-srv6

Introduction:

This document specifies Deterministic Networking data plane operation for SRv6 encapsulated user data.

4. DetNet Controller Plane Framework

Publication URL: https://tools.ietf.org/html/draft-malis-detnet-controller-plane-framework

Introduction:

This document provides a framework overview for the DetNet controller plane. It discusses concepts and requirements that will be basis for Detnet controller plane solution documents.

5. DetNet Configuration YANG Model

Publication URL: https://tools.ietf.org/html/draft-ietf-detnet-yang

Introduction:

This document contains the specification for Deterministic Networking flow configuration YANG Model. The model allows for provisioning of end-to-end DetNet service along the path without dependency on any signaling protocol.
